# Envoir brand

The Envoir mark is an **"@" whose inner glyph is an "e"** — *Envoir*, *email*, and *one keypair*
in a single simple shape. The `@` says mail and identity; the `e` names the product; the whole
thing is a clean, recognizable ring that scales from an app icon to a 16px favicon.

## Palette — "Aurora Indigo"

A deliberately **unique, email-appropriate** scheme. The blue-violet family carries *trust and
communication* (the reason mail/chat apps lean blue), the violet adds *sovereignty and premium*,
and the aurora-teal accent reads *fresh / new* — distinctive from the red/blue field of
incumbents.

| Token | Hex | Use |
|-------|-----|-----|
| Indigo | `#4C4DFF` | primary; gradient start |
| Violet | `#9A4DFF` | primary; gradient end |
| Brand gradient | `#4C4DFF → #9A4DFF` | the app tile, primary surfaces, CTAs |
| Aurora teal (accent) | `#00E0C7` | unread/new, notifications, success/go — used sparingly to pop |
| Ink | `#16132E` | text (a warm indigo-black, not pure black) |
| Paper | `#FBFAFF` | background (a faint violet-white) |

Neutrals are cool grays with a faint violet tint. Teal is the *only* accent — one pop, kept rare.

## Files

| File | Use |
|------|-----|
| `logo-mark.svg` | App-tile mark (Aurora Indigo gradient, 128×128, rounded tile). App icons, favicons. |
| `logo-mono.svg` | Single-color "@" mark via `currentColor` — light/dark UI, print, watermarks. |
| `wordmark.svg` | Mark + "Envoir" lockup for headers/navbars. |

## Usage

- Keep clear space ≈ the tile corner radius around the mark.
- Don't recolor the gradient, stretch, skew, or add effects. Use `logo-mono.svg` when one flat
  color is needed.
- The mark scales down cleanly; the favicon uses a slightly heavier stroke for small-size legibility.
